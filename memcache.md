memcached - win 安装

```
https://commaster.net/content/installing-memcached-windows
下载地址：
http://code.jellycan.com/files/memcached-1.2.5-win32-bin.zip
http://code.jellycan.com/files/memcached-1.2.6-win32-bin.zip
http://downloads.northscale.com/memcached-win32-1.4.4-14.zip
http://downloads.northscale.com/memcached-win64-1.4.4-14.zip【推荐使用】
http://downloads.northscale.com/memcached-1.4.5-x86.zip
http://downloads.northscale.com/memcached-1.4.5-amd64.zip
安装命令【管理员身份进入安装目录运行】：
memcached.exe -d install【安装成功可在 管理工具 - 服务 里面找到】
memcached.exe -d start【开启】
memcached.exe -d stop【关闭】
memcached.exe -d runservice -m 512
c:\memcached\memcached.exe -d uninstall【卸载】
```
